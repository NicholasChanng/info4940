Translate emotions into concise visual ideas that a novice can understand.

Examples:
- happiness -> rising sun, warm glow, bouncing circles
- love -> flowers, converging lines, blooming shapes
- loneliness -> one dim figure, empty space, muted palette
- outsider -> a single shape that moves differently from a crowd
- calm -> rain, slow ripples, soft gradients
- grief -> fading light, falling particles, heavy downward motion
- anxious -> jittering lines, tight spacing, unstable rhythm
- hopeful -> upward drift, brighter horizon, growing shapes
- anger -> sharp edges, hot colors, fast impacts

The metaphors should be visually specific enough to code but still poetic.
